# Engineered BCL for Esterification Reactions

This repository contains computational data for the engineering of BCL (butyrolactone biosynthesis-related enzyme) for improved esterification activity, supporting the 2026 fermentation research project.

## Related Publication

**Title:** Development of an Efficient Biocatalyst: Engineered Burkholderia cepacia Lipase for Aqueous Esterification

**Authors:** Tongtong Xu, Xiaojing Tian, Qiang Wang, Guanru Si, Jie Mei, Aobo Sha, Zhiming Rao, Wenqing Zhang, Xian Zhang

**Status:** Under review in *Fermentation*
